# Greetings Web Backend

This is the backend for the greetings web application.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Copy `.env.example` to `.env` and fill in your details:
   ```bash
   cp .env.example .env
   ```
3. Run in development mode:
   ```bash
   npm run dev
   ```

## Folder Structure

Based on an MVC-style layered architecture. See `src/` directory.
